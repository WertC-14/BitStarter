# Demo Flow

1. Open the Vercel URL.
2. Connect a Monad Testnet wallet (e.g. MetaMask pointed at Monad Testnet).
3. Create an investment campaign.
4. Open the campaign list and show the campaign.
5. Place an investment.
6. Show the updated funding progress.
7. Show the live activity feed.
8. Demonstrate usable withdrawal, weighted voting, final withdrawal, or protected refund.
9. Show the transaction hash.
10. Show GitHub Actions CI and local test output.
